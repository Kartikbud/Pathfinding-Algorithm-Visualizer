# A-Pathfinder-Algorithm-Visualizer

Algorithm visualizer for search-based path planning algorithms: DFS, BFS, A*. Built with pygame.

## Setup

```sh
python -m venv env_name
pip install -r requirements.txt
python main.py
```

## Usage

1. Click to place the start node.
2. Click to place the end node.
3. Click to draw obstacles.
4. Press Space to finalize the map.
5. Press `1` for DFS, `2` for BFS, `3` for A*.
